Design System

  • Chain Links: Symbolize connection between job seekers and opportunities
  • Circular Nodes: Represent interconnection points in the network
  • Magnifying Glass: Universal search icon in accent gold
  • SA Flag Colors: Emphasize South African market focus
  • Gradient Background: Modern, dynamic brand feel
  • Typography: Segoe UI, Arial, sans-serif

Technical Implementation

  • Format: SVG (Scalable Vector Graphics)
  • Advantages: Infinite scalability, small file size, crisp at any resolution
  • Browser Support: Chrome 80+, Firefox 41+, Safari 9+, Edge 79+
  • PWA Ready: Includes manifest.json for Progressive Web App support
  • Mobile Optimized: Apple touch icon and Android home screen icons
  • SEO Friendly: Proper Open Graph and meta tag integration
